# Project Name

Fishing Flotilla Management System

# Description

Это консольное приложение на языке C++ предназначено для управления рыболовной флотилией. Оно позволяет отслеживать информацию о траулерах, членах команды, рейсах и улове. Приложение включает функции аутентификации пользователей, управления данными (CRUD), аналитические отчеты и начисление бонусов членам экипажа. Данные хранятся в локальной базе данных SQLite.

# Installation

Для установки и запуска приложения необходимо:

1.  Убедиться, что в системе установлены компилятор C++ (например, GCC или Clang) и утилита `make`.
2.  Установить библиотеку `libsqlite3` и её заголовочные файлы. Например, в Ubuntu:
    ```bash
    sudo apt install libsqlite3-dev build-essential
    ```
    В macOS библиотека и заголовки обычно уже установлены как часть Xcode Command Line Tools (`xcode-select --install`).
3.  Склонировать репозиторий с исходным кодом:
    ```bash
    git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
    cd <НАЗВАНИЕ_КАТАЛОГА_РЕПОЗИТОРИЯ>
    ```
4.  Собрать проект. Предполагается, что в корне проекта есть `Makefile` (хотя в предоставленных файлах его нет, он необходим для команды `make`). Если `Makefile` отсутствует, сборка может выполняться вручную:
    ```bash
    g++ -std=c++17 -c mainn.cpp -o mainn.o
    g++ -std=c++17 -c db_manager.cpp -o db_manager.o
    g++ -std=c++17 -c auth.cpp -o auth.o
    g++ -std=c++17 -c flotilla_service.cpp -o flotilla_service.o
    g++ -o flotilla_app mainn.o db_manager.o auth.o flotilla_service.o -lsqlite3
    ```
    (Пример для GCC, точная команда зависит от структуры проекта и `Makefile`)
5.  При первом запуске приложение создаст файл базы данных `flotilla.db` и инициализирует его с помощью скрипта `database.sql`, который должен находиться в том же каталоге, что и исполняемый файл.

# Usage

1.  Запустить скомпилированное приложение из терминала:
    ```bash
    ./flotilla_app
    ```
2.  Приложение запросит логин и пароль.
    *   Для входа с правами администратора используйте логин `admin` и пароль `admin`.
    *   Для входа с правами члена команды используйте логины и пароли, созданные в таблице `CREW` базы данных (например, `cap_ivanov` / `123`).
3.  После успешной аутентификации появится меню с доступными операциями, такими как просмотр рейсов, улова, начисление бонусов и т.д.
4.  Следуйте инструкциям, выводимым в консоль, для ввода параметров операций.
